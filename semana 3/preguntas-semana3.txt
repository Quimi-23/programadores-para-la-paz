Pregunta 1 — Selección múltiple
¿Qué es Git?

R: B. Un sistema de control de versiones para registrar cambios en proyectos


Pregunta 2 — Selección múltiple
¿Cuál es la función principal de un repositorio?

R: B. Registrar y organizar los archivos y cambios de un proyecto


Pregunta 3 — Selección múltiple
¿Qué hace el comando git add .?

R: B. Agrega archivos al área de preparación para registrar cambios


Pregunta 4 — Selección múltiple
¿Qué hace el comando git commit?

R: A. Guarda un registro de cambios realizados en el proyecto
